# Roadmap

This roadmap tracks the current direction of Seedwork.

Completed work uses `✅`. Planned work uses `⚪`.

## Completed

- ✅ Local `llama-server` adapter with tool-calling support
- ✅ Domain schema and phased SQL migrations
- ✅ Staff-facing React dashboard for decisions, reporting, and corrective actions
- ✅ WhatsApp intake bridge via webhook handler
- ✅ n8n bridge for optional workflow automation
- ✅ Organization setup wizard and generated org profile flow
- ✅ Contributor documentation and public repo cleanup
- ✅ Public-facing README, FAQ, roadmap preview, and contribution guidance

## Near-Term

- ⚪ Slack, Telegram, and Discord messaging bridges
- ⚪ Broader commercial-domain templates beyond incubator/accelerator operations
- ⚪ Authentication and access control hardening for the staff UI
- ⚪ Deployment and infrastructure guides for broader self-hosting scenarios
- ⚪ Expanded admin workflows, reporting, and operational analytics

## Longer-Term

- ⚪ Additional communication and intake channels beyond current webhook flows
- ⚪ Stronger operator administration and governance tooling in the staff UI
- ⚪ More reusable organization templates for adjacent industries and operating models
- ⚪ Richer reporting, audit visibility, and quality-management workflows
- ⚪ Better onboarding and packaging for self-hosted deployments

## Contribution Targets Aligned To The Roadmap

If you want to contribute, the highest-leverage areas right now are:

- messaging bridges and intake channel expansion
- staff UI hardening and operator workflows
- deployment, setup, and self-hosting improvements
- domain adaptation for additional organization types
- reporting, analytics, and quality-management features
